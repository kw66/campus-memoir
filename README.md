## 项目简介

《模拟校园》是一个面向校园回忆录场景的网页游戏原型。  
先做一所学校，再逐步扩展到更多学校。

## 核心内容

| 模块 | 内容 |
| --- | --- |
| 地图 | 导入学校地图，移动小人，缩放和拖动 |
| 照片 | 添加照片点，路过时优先显示最近照片 |
| 建筑 | 命名、上传照片、设置入口，继续扩展室内空间 |
| 编辑 | 裁剪、旋转、绘制、替换、结构图编辑 |
| 存储 | 浏览器本地保存，不依赖后台 |

## 当前方向

- 先做“校园回忆录”
- 先做中国科学技术大学高新校区
- 之后扩展到更多学校
