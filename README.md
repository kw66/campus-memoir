# 模拟校园 / Campus Memoir

一款开源、非盈利、无需后台的校园回忆模拟游戏。

你可以导入自己的校园大地图，在地图上移动小人，添加照片点、建筑、寝室、实验室、食堂、体育馆、超市等回忆内容。项目先做“校园回忆录”模式，面向准毕业、已经毕业或想整理校园记忆的玩家；后续再扩展“校园进行时”模式。

## 当前功能

- 学校选择、新建和本地保存
- 导入校园大地图图片
- 地图缩放、拖动、玩家跟随和键盘移动
- 地图编辑器：底图裁剪、旋转、绘制、颜色替换、结构图绘制
- 结构对象：建筑、道路、桥、水体、场地、绿地、围墙、门等
- 游戏模式：照片点、附近建筑交互、建筑内部房间和人物原型
- 纯前端存储：浏览器 `localStorage` + `IndexedDB`

## 玩法方向

第一阶段重点是“校园回忆录”：

- 玩家在校园地图中移动
- 到达某个位置后添加照片点
- 过于接近的照片点会合并，避免地图上出现大量重复点
- 靠近照片点时优先显示照片
- 没有照片点时，才显示附近建筑、操场、篮球场等可交互对象
- 建筑可以命名、上传建筑照片、设置入口，未来继续扩展室内空间

长期目标是做成一个可以复用到多所学校的校园记忆编辑器和轻量游戏。

## 本地运行

可以直接打开 `index.html`。如果浏览器本地文件权限导致图片或存储行为异常，建议用本地静态服务器：

```powershell
Set-Location -LiteralPath C:\Users\KW\Desktop\campus-memoir
python -m http.server 8124 --bind 127.0.0.1
```

然后访问：

```text
http://127.0.0.1:8124/
```

## 数据说明

- 项目不需要后端。
- 学校列表和配置保存在浏览器本地。
- 地图图片、地图结构、游戏照片等内容也保存在当前浏览器环境中。
- 不同浏览器、`file://` 和 `http://127.0.0.1:8124/` 的存储空间不互通。
